# node-translate-bot

This is a Discord bot which provides a context menu to save translation preferences across guilds, then translates messages (to you only) when you request it

## Usage

Right click on a message, and go to Apps -> Translate Message

Click the button for which language you want to translate messages to, e.g. your native language

All discord servers with this bot will then translate to your native language the same way
